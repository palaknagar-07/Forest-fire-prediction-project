# 🔥 Forest Fire Prediction Project

A Machine Learning Flask web app that predicts Forest Fire Weather Index (FWI) based on meteorological parameters.

## 🚀 Tech Stack
- Python 3.11
- Flask
- Scikit-learn
- NumPy
- HTML

## 📊 Features Used
- Temperature
- RH
- Wind Speed
- Rain
- FFMC
- DMC
- ISI
- Classes
- Region

## 🛠 How to Run

```bash
git clone <repo>
cd Forest-fire-prediction-project
python3.11 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python application.py
